# Aero Squadron Mission Form App

This is a local desktop application for filling out Aero Squadron mission forms. It uses Python and Tkinter for the UI and generates PDFs using ReportLab.

## Key Files
- `AeroRoster20260127.json`: Member data
- `data/member_data.py`: Data handling logic
- `ui/desktop_ui.py`: UI implementation
- `main.py`: Entry point
